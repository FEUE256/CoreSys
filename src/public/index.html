CoreSys OS Edition

A visionary standalone operating system engineered by FÈUE

About

CoreSys OS Edition is a bold and independent operating system developed from scratch by FÈUE. Designed exclusively for new UEFI systems. The system now boots through a custom bootloader and enters protected mode without any BIOS support. This do not have any support for BIOS or EDK2 or 32 bit mode only 64 bit mode!
